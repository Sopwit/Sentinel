# Accessibility

Phase 51 adds persistent accessibility and productivity preferences.

## Supported Preferences

- Reduced motion.
- High contrast.
- UI density: Compact, Comfortable, or Large.

## Interaction Support

- Keyboard navigation for shell, Settings, dialogs, and Command Palette.
- Visible focus indicators through shared interaction tokens.
- Command Palette discoverability through `Ctrl/Cmd+K`.
- Screen-reader-oriented labels for primary QML controls.

## Boundary

Accessibility preferences affect presentation only. They do not alter provider routing, model
execution, permissions, memory behavior, exports, updates, cloud state, or task execution.
